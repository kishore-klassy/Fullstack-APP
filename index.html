AI Workflow Builder - Help

Getting Started

Build powerful AI workflows by connecting agents together, just like n8n!

  • Drag agents from the left sidebar to the canvas
  • Click and drag from output points (right side) to input points (left side) to create connections
  • Click agents to select them, then use Delete to remove them
  • Shift+Click on connections to delete them

Agent Types

Choose from various agent categories:

  • Data Processing: Collect, transform, and validate data
  • Machine Learning: Train models, make predictions, evaluate performance
  • Automation: Schedule tasks, send notifications, monitor systems
  • Analysis: Analyze data, create visualizations, generate reports
  • Communication: Send emails, post to Slack, call webhooks
  • Triggers: Start workflows with timers, file watchers, or API calls

Workflow Execution

The system automatically determines execution order based on connections:

  • Agents execute in topological order (dependencies first)
  • Validation checks for circular dependencies and disconnected agents
  • Real-time visual feedback during execution
  • Detailed execution logs in the right panel

Keyboard Shortcuts

  • Ctrl+S Save workflow
  • Ctrl+O Load workflow
  • Ctrl+R Run workflow
  • Delete Delete selected agent
  • Escape Cancel connection or deselect
  • Shift+Click Delete connection

Tips & Tricks

  • Use trigger agents (Timer, File Watcher, API) to start workflows
  • Validate your workflow before running to catch issues
  • Save frequently to preserve your work
  • Connection lines show data flow direction with arrows
  • Agents glow when selected or during execution
